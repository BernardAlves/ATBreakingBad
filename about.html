ASSESSMENT - API Breaking Bad

Sobre o projeto

Como proposto pelo assessment, esse é um site responsivo que faz uso de dados de alguma API. O site responde a todas as telas, e abaixo de de width:650px, há um breakpoint para o mediaquery, chamado de styles2.css, para um uso mais eficiente de uma tela menor. Sobre a API, usei uma para fazer a busca de personagens, e decidi usar outra API para a busca de episódios, que estava mais completa.
